Setting up ONVIF cameras and adding them to Unifi Protect.     

This projects shows how I integrated my Hikvision Cameras into Unifi Protect.       

What I used

  • Unifi Cloudkey Gen 2+ 

What I did to integrate the cameras    

I wanted to integrate Hikvision cameras into the Unifi Protect network so I began researching on how this is possible. I wanted to switch over to Unifi Protect because the UI is very nice and easy to navigate. The Hikvision app is also really bad so I wanted it to be easy to view our security cameras for my parents. To setup ONVIF mode for the cameras I had to unplug the Ethernet ports going into my DVR and move it into a POE Switch. It powered all 4 of my cameras and allowed me to expose the cameras to the main network. Now to create a seamless security system I wanted to have a backup DVR (if main was ever destroyed) and a Main DVR. This prevents intruders from destroying camera footage. I also hid the CloudKey Gen 2+ on my desk as it would look like a different device.

Now, to add a backup and main dvr, I connected the Hikvision DVR to the Hikvision Cameras via ONVIF which now required a password and username. Now I had to setup my new Cloud Key by creating and linking the box to my Unifi Account which binds the device so even if its stolen it is not usable to the person. Now I had to install the Protect application which took about 10 mins. After that I could now format the HDD so it would be ready to upload video footage 247 to it. After linking the cameras to the Cloud Key I had to setup motion detection and since the cameras are 3rd party I had to find my own way to create motion notifications.

Since my cameras are 3rd party I used Home Assistant to connect my ONVIF cameras and then check for motion detection via the cameras sensors. I then created a automation that would send an webhook to my Unifi Cloudkey that would send a motion alert from the app. I had to do this for each camera. That's how I created a robust smart security system that I could built on to create a Smart Home.
