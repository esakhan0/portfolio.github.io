I built a portfolio using Dreamweaver

I wanted a clean portfolio that loads fast, works on GitHub Pages, and is easy to update.

What I focused on

  • Consistent design system
  • Clean file structure
  • Fixed background
  • Premium feel
  • Zero cost

Biggest lesson

Consistency makes a site instantly professional. One shared style file controlled fonts, layout, spacing, and background behaviour across every page.

Experience with Dreamweaver

Dreamweaver worked perfectly with GitHub Pages. Using git pull and git push let me sync changes across machines easily.
