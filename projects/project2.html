I made a Minecraft server that my friends can play on!

I wanted a minecraft server that can be hosted 24/7 and was available to my friends anytime.

Items I used!

  • Mini PC 
  • Sodimm Ram 8GB x 2 
  • USB - Ubuntu .iso 
  • MCSM - software 

Problems & Solutions

I wanted to make a minecraft server with a panel for friends to play on anytime. But as usual I ran into many problems. I ordered a cheap Lenovo Thinkcentre which I would repurpose as a server. When it arrived, I ran into a problem, it started beeping when I tried to start it up and after researching I found out it was a RAM issue. After opening the computer up, I found that the RAM was faulty so I bought 2 sticks or 8GB of SODIMM RAM and once it arrived I quickly ran to go apply it.         

 After I applied the RAM to the computer, I started it up and it didn't even beep and the storage led started to blink. It then booted right into windows (It came with windows installed)! Now I was able to install Ubuntu 24.04.3 LTS by using a USB stick and adding a .iso file to install ubuntu. I put the USB stick in the the mini computer but it still booted into windows. So I went into the computers BIOS and selected the boot order to be my USB stick first and then the HDD. Then finally it booted right into the Ubuntu Installation Menu where there I could install Ubuntu on my HDD permanently.                   

After getting Ubuntu ready 

After installing Ubuntu successfully I researched on some popular minecraft server management softwares available. After many hours I found a really nice software with beautiful UI and a good system to work with. It was called MCSM. MCSM stands for Minecraft Server Manager and it was quick and easy to install it as a SystemCTL service. After installing I was able to access the web ui at my servers address on port 23333. After that I created a user for myself and installed the package for the Minecraft Version 1.21.4 Paper. 

1.21.4 Paper is a version on Minecraft that support plugins (allow the game to function differently without players downloading anything) and once installed I could boot it up. I used a variety of plugins such as TAB, CMI, etc. These plugins would help me to make the server look nice and professional. I also used Multiverse-Core (Plugin) to help me to make different worlds running on one server so that I could create a spawn world and the main game world.                

Making it easy to access to others

To allow friends to moderate and manage the server with me, I installed Cloudflare Tunnels on my mini computer which would essentially authorise  my friends to type a URL in google and access the panel. This would allow me to forward ports without exposing through my network. Since I was using Cloudflare tunnels to forward my Game (Port 25565) and my Panel the control the server (Port 23333) it would mean that my network would not be vunreable to attacks such as DDOS because Cloudflare proxies my tunnels and also helps to prevent DDOS attacks by blocking IP's with mass amount of requests to the server. Also to increase server security I used VLAN'S to isolate and protect my other devices from my server. This helped extensively and gave me an extra sense of reassurance and used containers to isolate my minecraft instance from the main server.
