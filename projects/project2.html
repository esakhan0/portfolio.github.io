I made a minecraft server that my friends can play on!

I wanted a minecraft server that can be hosted 24/7 and was available to my friends anytime.

Items I used!

  • NEW RAM STICKS 8GB X 2
  • USB with Ubuntu installation media

My Experience!

I wanted to make a minecraft server with a panel for friends to play on anytime. But as usual I ran into many problems. I ordered a cheap thinkcentre which I would repurpose as a server. When it arrived, it started beeping ‚Äî the RAM was faulty. I ordered 8GB x 2 DDR3 SODIMM sticks, installed them, and the server booted perfectly.

Adding a node instance
